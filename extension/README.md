# Hype Drop — Queue Watcher (browser extension)

A zero-server browser extension that watches Pokémon Center (and any other
[Queue-it](https://queue-it.com/) store) for a virtual queue and alerts you
**in your browser** with a desktop notification, a toolbar badge, and an
optional sound. No server, no email, no proxy.

## Why this beats the server approach

Pokémon Center is behind **Akamai bot protection**, which reliably blocks
automated server-side checks. This extension runs inside *your* browser, so
every check uses your **real IP, real browser fingerprint, and your real
Akamai session cookies** — exactly what gets past the block. It detects the
queue two ways:

- **Content script** — when you open a Pokémon Center / Queue-it page, it reads
  the real, already-rendered page (ground truth) and reports the state.
- **Background timer** — every couple of minutes it checks your watched URLs
  from your browser and notifies you if a queue appears, even with no tab open.

## Install (Chrome / Edge / Brave — unpacked)

1. Open `chrome://extensions` (or `edge://extensions`).
2. Turn on **Developer mode** (top-right).
3. Click **Load unpacked** and select this `extension/` folder.
4. Pin the Hype Drop icon to your toolbar. Click it to open the popup.

That's it — it starts watching `https://www.pokemoncenter.com/` right away.

## Using it

- **Popup** — see each watched site's status (🟢 open · 🔴 queue · 🟠 blocked),
  add/remove URLs, pause/resume, and hit **Check now**.
- Add the specific **product/drop URLs** you care about — queues are often
  applied per page during a drop, not just on the homepage.
- **Settings** — check interval (minutes), sound on/off, and whether to notify
  when a queue starts and/or ends.
- **Test alert** — fires a sample notification so you can confirm notifications
  (and sound) are allowed by your OS/browser.

### Tips

- Keep the browser running (a background window is fine) so the timer can check.
- If your OS suppresses notifications, enable them for your browser in the OS
  notification settings, and allow the extension to show notifications.
- Watching a site other than Pokémon Center? Add its URL in the popup — the
  extension will request one-time permission for that host.

## How detection works

The shared classifier (`detector.js`, unit-tested in `../test/`) flags a queue
on any of: a `*.queue-it.net` URL, `QueueITAccepted-*` cookies, Queue-it scripts
in the page, or waiting-room text ("you are now in line", "estimated wait time",
…). A `403`/anti-bot page is reported as **blocked**, never as a false "open".

## Files

| File | Role |
|------|------|
| `manifest.json` | MV3 manifest (permissions, background, content script) |
| `background.js` | service worker: timer, background checks, notifications, badge |
| `content.js` | reads real Pokémon Center / Queue-it pages you visit |
| `detector.js` | shared open/queue/blocked/error classifier |
| `popup.html/js/css` | status view + configuration |
| `offscreen.html/js` | plays the alert tone (service workers can't play audio) |
| `icons/` | toolbar icons (generated by `scripts/generate-extension-icons.js`) |

## Privacy

Everything stays local. The extension talks only to the store URLs you add
(to check them) and stores its settings in your browser. It has no backend and
sends your data nowhere. It only *notifies* — it never carts or checks out.
